MovieVault is a Spring Boot based RESTful application for managing movies. It provides full CRUD (Create, Read, Update, Delete) operations with a clean layered architecture.

## Features
- Add new movies
- View all movies
- Update existing movie details
- Delete movies
- REST API endpoints
- MySQL database integration
- Swagger API documentation

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- Swagger

## Project Structure
- Controller Layer
- Service Layer
- Repository Layer
- Entity Layer

## How to Run the Project

1. Clone the repository
    https://github.com/Saifulla-dev/MovieVault.git
2. Set environment variables
    DB_USERNAME=your_db_username 
    DB_PASSWORD=your_db_password
3. Run the project using:
    mvn spring-boot:run
4. Open Swagger UI:
    http://localhost:90/swagger-ui.html
